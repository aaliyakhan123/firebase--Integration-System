# Inventory Tracking System

## Project Overview

The Inventory Tracking System is a web application developed using **React.js**, **Redux Toolkit**, and **Firebase Realtime Database**.
This application allows users to manage product inventory by adding, viewing, updating, and deleting product records in real time.

The project demonstrates how modern frontend technologies can be integrated with a cloud database to build a simple inventory management system.

## Features

* Add new products with name, category, price, and stock
* View all products in a list
* Update product information
* Delete products from inventory
* Real-time data synchronization using Firebase

## Technologies Used

* React.js
* Redux Toolkit
* Firebase Realtime Database
* JavaScript
* HTML
* CSS

## How It Works

The application uses Redux Toolkit for state management.
Firebase Realtime Database is used as the backend to store and retrieve product data.

When a user adds a product, the data is sent to Firebase and stored in the database.
The application then fetches the updated product list and displays it instantly on the user interface.
